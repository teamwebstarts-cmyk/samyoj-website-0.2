# samyoj-website-0.2

Offline mirror of the AiSensy marketing website, curated into a clean static site.

## What's inside
- `index.html` — sitemap / hub page (open this to browse the site)
- `aisensy.com/` — the website pages (home, pricing, features, industries, integrations, case-studies, tutorials, legal, etc.)
- `umsousercontent.com/`, `img.youtube.com/`, `d3jt6ku4g6z5l8.cloudfront.net/`, `cdnjs.cloudflare.com/` — local copies of images/assets referenced by the pages
- `.nojekyll` — disables GitHub Pages Jekyll processing

## Notes
- Pages use inline CSS, so each page is styled on its own.
- Google Fonts are loaded from the network; everything else is local.
- This is a static snapshot for personal/archival use.

## Local browsing
Open `index.html` in a browser, then click "Open Homepage" or any section.

## Deploy
Published with GitHub Pages from the `main` branch (root).
