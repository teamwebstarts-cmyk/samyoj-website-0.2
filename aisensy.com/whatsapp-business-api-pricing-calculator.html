    FAQ

    Frequently asked questions about WhatsApp Business API Pricing

    What changed in WhatsApp API pricing in 2026?

    Meta moved WhatsApp Business API billing from per-conversation to per-message pricing, effective 1 January 2026. Earlier, one 24-hour conversation window carried one charge regardless of how many template messages it held. Now each marketing, utility and authentication message is charged individually. Service conversations remain free, and the calculator above already runs on the per-message model.

    Which is the cheapest WhatsApp API?

    The cheapest way to start is a free plan. AiSensy's Free Forever plan carries a ₹0 platform fee and zero API procurement fee. Meta's per-message charges apply on every provider, so the real comparison is platform fee plus per-message rate. Use the calculator above to price your exact volume.

    How to use WhatsApp Business API for free?

    Start on AiSensy's Free Forever plan: the platform fee is ₹0 and the WhatsApp Business API application costs nothing. You then pay only Meta's per-message charges for the template messages you send, and replies inside the 24-hour service window are free, so support conversations cost nothing.

    Is WhatsApp OTP API free?

    No. OTPs are authentication messages, and Meta charges them per message. In India the authentication rate is ₹0.145 per message. There is no separate OTP licence fee, and on AiSensy the API access itself costs nothing, so your OTP cost is simply volume multiplied by the per-message rate.

    What is the 24-hour rule for WhatsApp Business?

    When a customer messages you, a 24-hour service window opens. Every reply you send inside that window is free and needs no template. Once the window closes, reaching the customer again requires a pre-approved template message, charged at Meta's per-message rate for its category. This is why WhatsApp support costs nothing to run.

    What is the difference between WhatsApp API and WhatsApp Business API?

    They are the same thing; both names refer to Meta's WhatsApp Business Platform. The real distinction is between the free WhatsApp Business app, built for small teams on one phone, and the API, which connects through a provider like AiSensy for bulk campaigns, automation, integrations and team inboxes.

    Q. How to get blue tick on WhatsApp Business for free?

    The blue tick, now the Meta Verified badge for the API, is applied for through your provider. AiSensy submits your blue tick application at no charge on every plan. Meta decides approval based on your business's notability and documentation, so the application is free, and the outcome rests with Meta.

    Which WhatsApp API has no Meta approval?

    None that is safe to use. Every legitimate WhatsApp Business API connection runs through Meta, directly or via a verified Business Solution Provider. Unofficial tools that skip Meta approval work by automating regular WhatsApp, which violates WhatsApp's terms and commonly ends in the number being banned. Use a verified provider.

    Does GST apply to WhatsApp API pricing?

    Yes. A flat 18% GST applies for Indian users on top of both the platform subscription and message charges. The calculator's estimate excludes GST, and final message billing runs through prepaid WhatsApp Conversation Credits at the live per-country rate.
